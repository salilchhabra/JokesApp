A single-screen mobile application that:
Every minute takes a joke from an open API (https://geek-jokes.sameerkumar.website/api?format=json)
Adds it (the joke) to the RecyclerView at the end
Maximum of 10 jokes on the list
The new jokes replace old ones
When you open the app again, all old jokes should be displayed
Works on devices starting with Android SDK 21
Use these in development:
Kotlin
Coroutines
Retrofit 2
Koin
Clean architecture approach
